Web Performance
影响网络性能的因素有很多，如网络延迟。当请求一个网页或网页相关的资源，浏览器新建一个线程发起请求和初始化DNS Server，DNS查找。建立TCP/IP连接的时候，需要进行三次握手。TCP握手，像通信协议一样，只是使用标准化方式定义不同部分的交流方式。
造成网络延迟的另一原因是加载网页所需组件所需的HTTP请求数。页面所需的资源如图片、外部CSS、Javascript文件，都需要从服务器端加载。每次请求将新起一个线程，每次请求过程都包括DNS lookup、TCP连接、HTTP请求和响应。
浏览器解析和渲染页面也是影响网络性能的另一个因素。大多数的现代浏览器拥有以下架构:地址栏、历史操作、渲染引擎、Javascript引擎和用于处理HTTP请求的网络层。
